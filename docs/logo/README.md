# Logo

- Created the [logo.png](logo.png) using round shapes and stairs using Canva pro. Downloaded the transparent PNG file.
- Created the [favicon.ico](favicon.ico) from the PNG file using [favicon.io](https://favicon.io/favicon-converter/)
- Converted the ![logo.svg](logo.svg) from the PNG file using [convertio.co](https://convertio.co/png-svg/)
- Used ChatGPT Codex to craete the [brand-logo.tsx](brand-logo.tsx) supporting light and dark mode using the SVG file.
