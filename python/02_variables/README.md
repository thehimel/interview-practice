# Variables

## Problem Statement

Given the list `["Apple", 2, 4.2]`:

1. Unpack the list into three variables: `name`, `quantity`, and `weight`.
2. In another function, return the types of `name`, `quantity`, and `weight`.
3. Assign the values of the list elements to global variables `X`, `Y`, and `Z` (first element to `X`, second to `Y`, third to `Z`).

**Requirements:**

- Unpack `["Apple", 2, 4.2]` into `name`, `quantity`, `weight`
- Create a function that returns the types of `name`, `quantity`, and `weight`
- Assign the first element to global `X`, second to `Y`, third to `Z`

## Example

**Input:** `["Apple", 2, 4.2]`

**Output (types):** `(str, int, float)`

**Global variables after assignment:** `X == "Apple"`, `Y == 2`, `Z == 4.2`
