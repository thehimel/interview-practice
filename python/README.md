# Python Exercises

## How to Perform an Exercise

1. **Choose a topic**: Each directory is a separate exercise.
2. **Read the problem**: Open the topic’s `README.md` for the problem statement and requirements.
3. **Implement in `exercises/<topic>.py`**: Edit the topic file in the topic's `exercises/` directory and implement the required logic.
4. **Run tests**: Run tests in `exercises/` to verify that exercise.

## Solution

- Each topic includes a `solutions/` directory with a reference implementation.
- Run tests in `solutions/` to verify the solution.
- By default, `pytest` runs solution tests only (exercise tests are ignored).

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```
