# Python Exercises

## How to Perform an Exercise

1. **Choose a topic**: Each directory is a separate exercise.
2. **Read the problem**: Open the topic’s `README.md` for the problem statement and requirements.
3. **Implement in `exercise.py`**: Edit the `exercise.py` file in that topic directory and implement the required logic.
4. **Run tests**: Run `test_<topic>_exercise.py` to test that exercise.

## Solution

- Each topic includes a `solution.py` file with a reference implementation.
- Run `test_<topic>_solution.py` to verify the solution.
- By default, `pytest` runs solution tests only (exercise tests are ignored).

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```
